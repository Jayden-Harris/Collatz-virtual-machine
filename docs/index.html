What is CAL?

CAL (Collatz Assembly Language) is a custom low-level assembly-style language designed for symbolic experimentation. It compiles to bytecode that runs directly on the CVM.

It's ideal for exploring control flow, register operations, and system-level logic in a compact and novel form.

CAL Documentation

What is CVM?

CVM (Collatz Virtual Machine) is a lightweight virtual CPU and runtime environment for executing CAL programs. It features a register set, a syscall interface, and deterministic instruction processing based on Collatz-inspired logic.

Understanding CVM helps you grasp how CAL operates under the hood.
