Welcome to CAL

CAL (Collatz Assembly Language) is a custom low-level language designed to run on the Collatz Virtual Machine (CVM).

What is CAL?

CAL is a minimalist assembly-like language inspired by the Collatz function and symbolic computation. Programs written in CAL are parsed into tokens, assembled into bytecode, and executed by the CVM.

Features

  • Simple syntax
  • Custom instruction set
  • Register-based design
  • Syscall integration
